# 📚 BookStoreApp

A complete MERN stack bookstore application with:

- User Authentication
- Cart & Wishlist
- Order Management
- Sell Books Feature
- Dark Mode UI

## 🚀 Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB

## ▶️ Run Locally

### Backend:
cd backend
npm install
node server.js

### Frontend:
cd frontend
npm install
npm start
